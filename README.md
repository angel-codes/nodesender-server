# Node Sender - NodeJS / Express

Node Sender application server built with JavaScript and Express

## Setting up

To run this project on your computer, you must follow the following steps. You need to have NodeJS v12.18.0+ installed.

- Run the following command in your terminal/cmd to install all of the dependencies of the project.
  ```sh
  yarn install
  # or
  npm install
  ```
- Then, you can run the server using the following script:
  ```sh
  yarn start
  # or
  npm start
  ```

## This project use

- JavaScript
- NodeJS
- ExpressJS
- Mongoose / MongoDB
- Multer
- JsonWebToken
- Argon2
